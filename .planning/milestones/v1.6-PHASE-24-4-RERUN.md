## §3-event terminal taxonomy verified

After 24.4 lands, the SSE wire protocol has three terminal event
taxa, each mapping to a distinct architectural truth:

| Event | Architectural meaning | When emitted |
|---|---|---|
| `event: done` | Model-decided completion | model returns terminal response with no tool_calls |
| `event: orchestration_terminated` | Policy-decided completion | K-fold canonical-recurrence trigger fires |
| `event: error` | Transport / runtime failure | timeout / budget / recursive call / tool loop error |

The taxonomy itself encodes the architectural decomposition. Consumers
reading SSE event names branch on architectural truth, not heuristics.
