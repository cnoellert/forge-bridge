# Phase 15 (FB-C): LLMRouter Tool-Call Loop — Verification Report

**Phase Goal:** `LLMRouter.complete_with_tools(prompt, tools, sensitive=...)` runs the full agentic loop — send prompt + tool schemas, parse `tool_call`, execute each tool against the registered MCP tools in-process, feed result back, repeat until the LLM returns a terminal response. Provider-neutral coordinator + thin Anthropic + Ollama adapters. Sensitive routing preserved verbatim from `acomplete()`. Hard iteration cap, wall-clock cap, repeat-call detection, ingest-time result truncation, sanitization boundary on tool results, and a recursive-synthesis guard ship in the same phase.
