Create `forge_bridge/learning/execution_log.py` with:

1. Module-level constants:
   - `LOG_PATH = Path.home() / ".forge-bridge" / "executions.jsonl"`
   - Import: `ast, hashlib, json, logging, os, textwrap` from stdlib, `dataclasses.dataclass`, `typing.Optional`, `pathlib.Path`, `datetime.datetime, timezone`

2. `_LiteralStripper(ast.NodeTransformer)` class:
   - `visit_Constant(self, node)`: return `ast.Constant(value="STR")` for strings, `ast.Constant(value=0)` for int/float, pass through otherwise

3. `normalize_and_hash(code: str) -> tuple[str, str]` function:
   - `textwrap.dedent(code).strip()`, then `ast.parse` -> `_LiteralStripper().visit(tree)` -> `ast.unparse`
   - On `SyntaxError`: use the dedented/stripped code as-is
   - Return `(normalized_code, hashlib.sha256(normalized.encode()).hexdigest())`

4. `ExecutionLog` class:
   - `__init__(self, log_path: Path = LOG_PATH, threshold: int = 3)`:
     - `self._path = log_path`
     - `self._threshold = int(os.environ.get("FORGE_PROMOTION_THRESHOLD", threshold))`
     - `self._counters: dict[str, int] = {}`
     - `self._promoted: set[str] = set()`
     - `self._code_by_hash: dict[str, str] = {}`
     - `self._intent_by_hash: dict[str, Optional[str]] = {}`
     - Call `self._replay()`
   - `_replay(self) -> None`: read each line of JSONL, rebuild `_counters`, `_code_by_hash`, `_intent_by_hash`, and `_promoted` set from `rec.get("promoted")` entries. Wrap in try/except for malformed lines.
   - `record(self, code: str, intent: Optional[str] = None) -> bool`:
     - Call `normalize_and_hash(code)` to get `(normalized, h)`
     - Update `_code_by_hash[h] = code` and `_intent_by_hash[h] = intent`
     - Increment `_counters[h]`
     - Build record dict: `{"code_hash": h, "raw_code": code, "intent": intent, "timestamp": datetime.now(timezone.utc).isoformat(), "promoted": False}`
     - `self._path.parent.mkdir(parents=True, exist_ok=True)`
     - Append with `open(self._path, "a")` -> `fp.write(json.dumps(rec) + "\n")` -> `fp.flush()`
     - Return `True` if `_counters[h] >= self._threshold and h not in self._promoted`, else `False`
   - `mark_promoted(self, code_hash: str) -> None`:
     - Add to `self._promoted`
     - Append `{"code_hash": code_hash, "promoted": True, "timestamp": ...}` to JSONL
   - `get_code(self, code_hash: str) -> Optional[str]`: return `self._code_by_hash.get(code_hash)`
   - `get_intent(self, code_hash: str) -> Optional[str]`: return `self._intent_by_hash.get(code_hash)`
   - `get_count(self, code_hash: str) -> int`: return `self._counters.get(code_hash, 0)`
