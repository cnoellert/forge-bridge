1. Imports: `functools, logging, os` from stdlib; `pathlib.Path`; `typing.TYPE_CHECKING, Callable`

2. Constants:
   ```python
   logger = logging.getLogger(__name__)
   SYNTH_DIR = Path.home() / ".forge-bridge" / "synthesized"
   QUARANTINE_DIR = Path.home() / ".forge-bridge" / "quarantined"
   ```

3. `ProbationTracker` class:
   - `__init__(self, failure_threshold: int | None = None, synth_dir: Path = SYNTH_DIR, quarantine_dir: Path = QUARANTINE_DIR)`:
     - `self._threshold = failure_threshold if failure_threshold is not None else int(os.environ.get("FORGE_PROBATION_THRESHOLD", "3"))`
     - `self._successes: dict[str, int] = {}`
     - `self._failures: dict[str, int] = {}`
     - `self._synth_dir = synth_dir`
     - `self._quarantine_dir = quarantine_dir`

   - `record_success(self, tool_name: str) -> None`:
     - `self._successes[tool_name] = self._successes.get(tool_name, 0) + 1`

   - `record_failure(self, tool_name: str) -> bool`:
     - `self._failures[tool_name] = self._failures.get(tool_name, 0) + 1`
     - Return `self._failures[tool_name] >= self._threshold`

   - `wrap(self, fn: Callable, tool_name: str, mcp) -> Callable`:
     - Return an async wrapper using `@functools.wraps(fn)` that:
       - On success: calls `self.record_success(tool_name)`, returns result
       - On exception: calls `should_quarantine = self.record_failure(tool_name)`, if True calls `self.quarantine(tool_name, mcp)`, then re-raises

   - `quarantine(self, tool_name: str, mcp) -> None`:
     - `src = self._synth_dir / f"{tool_name}.py"`
     - If src exists: `self._quarantine_dir.mkdir(parents=True, exist_ok=True)`, `src.rename(self._quarantine_dir / f"{tool_name}.py")`
     - Log warning with quarantine path
     - Try `mcp.remove_tool(tool_name)`, catch Exception and pass

   - `get_stats(self, tool_name: str) -> dict[str, int]`:
     - Return `{"successes": self._successes.get(tool_name, 0), "failures": self._failures.get(tool_name, 0)}`
