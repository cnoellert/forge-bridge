2. Insert a salvage block between line 419 and line 421 of `OllamaToolAdapter.send_turn()`:
   ```python
   if not tool_calls and text:
       salvaged = _try_parse_text_tool_call(text)
       if salvaged is not None:
           tool_calls.append(salvaged)
           text = ""  # consumed — don't double-emit as terminal content
   ```
3. The helper must:
   - Try `json.loads(text)`; on `JSONDecodeError`, return `None`.
   - Verify the parsed object is a dict with `name` (str) and `arguments` (dict) keys; return `None` otherwise.
   - Construct a `_ToolCall(ref=f"0:{name}", tool_name=name, arguments=dict(arguments))`.
4. After the fix, `test_text_content_tool_call_salvaged` flips PASSED, `test_plain_text_terminal_response_not_misclassified` STAYS PASSED, and 16 of 16 tests in `test_ollama_adapter.py` are green.
