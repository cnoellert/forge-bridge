  <action>
Create forge_bridge/mcp/registry.py with these exact components:

1. Module-level constants:
   ```python
   _SYNTH_RESERVED_PREFIXES: frozenset[str] = frozenset({"synth_"})
   _VALID_PREFIXES: frozenset[str] = frozenset({"flame_", "forge_", "synth_"})
   ```

2. `_validate_name(name: str, source: str) -> None`:
   - If name does not start with any prefix in _VALID_PREFIXES: raise ValueError with message f"Tool name {name!r} must start with flame_, forge_, or synth_. Got source={source!r}."
   - If source != "synthesized" and name starts with any prefix in _SYNTH_RESERVED_PREFIXES: raise ValueError with message f"Tool name {name!r} uses a reserved synth_ prefix. Only the synthesis pipeline may register under synth_."

3. `register_tool(mcp: FastMCP, fn: Callable, name: str, source: str, annotations: dict | None = None) -> None`:
   - Call _validate_name(name, source)
   - Call mcp.add_tool(fn, name=name, annotations=annotations, meta={"_source": source})

4. `register_tools(mcp: FastMCP, fns: list[Callable], prefix: str = "", source: str = "user-taught") -> None`:
   - For each fn in fns: compute name = f"{prefix}{fn.__name__}" if prefix else fn.__name__
   - Call register_tool(mcp, fn, name=name, source=source)
